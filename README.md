## 🚀 Features

- **📈 KPI Cards** — Real-time display of Sales Target, Sales Amount, Variance, % Achievement, and Units Sold
- **🗺️ Regional Analysis** — Target Met % breakdown by region (East, West, North, South)
- **🍩 Target Status Donut Chart** — Visual split between Met vs. Not Met targets
- **📊 Combo Chart** — Sales Target vs Sales Amount vs % Age Achievement per region
- **📉 Variance by Product Category** — Line chart across Books, Clothing, Electronics, Food & Furniture
- **🔽 Dynamic Slicers** — Filter by Year, Month, Target Status, Region, and Product Category
